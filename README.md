# Digital Attendance App

## Overview

Digital Attendance App is a Flutter application developed as part of the Mobile App Development Internship.

## Features

- Add Students
- Edit/Delete Students
- Register Subjects
- Take Attendance
- View Attendance History
- Lecturer Profile
- Local Storage using SQLite
- SharedPreferences for Lecturer Information
- Responsive UI

## Technologies Used

- Flutter
- Dart
- SQLite
- SharedPreferences

## Setup

1. Clone or extract the project.
2. Open the project in Android Studio or VS Code.
3. Run:

flutter pub get

4. Connect an Android device or emulator.

5. Run:

flutter run
